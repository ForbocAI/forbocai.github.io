# Aesthetic Protocols: Cozy Storybook Technology

## Directive

This site is the active public face of ForbocAI. It should feel warm, imaginative, and welcoming.

The old monochrome / grimdark system has been preserved in `Forboc.AI/monochrome/`. Do not pull that aesthetic back into the main brochure site.

Keep the main Layer 1 company framing clearly about ForbocAI, Inc. The visual system may diverge from `monochrome/`, and the copy does not need to mirror it line for line, but the cozy site still needs to cover the same feature surface.

## 1. Tone

- warm
- playful
- polished
- hopeful
- handcrafted
- story-forward

Public visitors should feel invited into a living creative world, not dropped into a hostile void transmission.

## 2. Visual Language

Prefer:

- lantern light
- mossy greens
- honey golds
- soft sky blues
- bark browns
- petal pinks
- rain-glow reflections
- rounded silhouettes
- stitched or carved details

Avoid:

- glitch noise as the core identity
- scanline obsession
- void symbolism
- oppressive black-on-black severity
- horror framing

## 3. Typography

- choose expressive, human-feeling type
- prioritize readability and warmth
- use display typography sparingly and intentionally
- avoid making the site feel like a corrupted terminal

## 4. Motion

- subtle drifting motion
- gentle reveals
- floating spores, lantern shimmer, leaf sway, or rain ripple style cues
- motion should feel alive, not broken

## 5. Copy Guidance

Use language that is:

- clear
- inviting
- imaginative
- confident

Use `monochrome/` as a reference for company scope, not as a template for exact wording. The active site should:

- stay clearly company-level rather than game-level
- keep the product framing focused on living character layers, memory, Souls, Ghost Scouts, and the `$FAI` market path
- maintain public clarity for partners and investors
- preserve the cozy tone without drifting into monochrome imitation
- prefer fairy-story warmth over hard technical cadence on the homepage

Do not lean on:

- cryptic machine jargon
- cruelty
- nihilism
- blood ritual metaphors
- "we are already dead" energy

## 6. Product Framing

Layer 1 should present ForbocAI as:

- technology for living NPCs
- memory and continuity for game characters
- guidance and support for autonomous worlds
- creative infrastructure for studios

Keep deep implementation details out of the brochure site. Use metaphor lightly and explain real value clearly.

## Rule

If a design choice feels like it belongs to the preserved `monochrome` fork aesthetically, it probably does not belong on `forboc.ai`.

If a messaging choice starts to sound like a different company, or sounds like a game landing page instead of a company brochure, treat that as a warning and correct it.
