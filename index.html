The Substrate

A local-first, privacy-focused architecture that merges rigourous ECS game loops with fluid Large Language Models.

Generative NPC Cortex

Powered by fine-tuned SLMs (SmolLM2) and edge-resident RAG buffers. Our "Micro-NPC" architecture enables low-latency, context-aware dialogue with persistent long-term memory, running entirely on-device.

ECS-Redux Hybrid Engine

A deterministic backbone. We decoupled state from behavior using a strictly functional Entity-Component-System pipeline, ensuring that every AI decision tree is replayable, auditable, and bug-free.

Ghost Agent QA

Autonomous testing pipelines that play the game before humans do. Our "Ghost Agents" rigorously audit procedurally generated capabilities using Playwright and Vitest, ensuring zero dead-ends.

Future Protocol: The Agent Economy

We are architecting a decentralized framing for autonomous agent interoperability. A tokenized layer where your unique NPC logic and memory states can be owned, traded, and utilized across compatible worlds.
